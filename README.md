# Silat Scoring Platform

A multi-platform Pencak Silat scoring system with:
- Flutter judge app
- web admin application
- live scoring engine
- Stripe billing
- Firebase authentication

## Structure

- `apps/` → frontend applications
- `services/` → backend services
- `packages/` → shared rules and types
- `legacy/` → previous codebases for reference
- `infra/` → deployment and integration config
- `docs/` → architecture and project documentation
